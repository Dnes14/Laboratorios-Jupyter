problema = pulp.LpProblem("utilidades",pulp.LpMaximize)

# Variables de decisin y marcamos la no negatividad

x1 = pulp.LpVariable("x1",lowBound = 0)
x2 = pulp.LpVariable("x2",lowBound = 0)

# Funcin objetivo
problema += 5*x1 + 4*x2

#Restricciones
problema += 6*x1 +  4*x2 <= 24
problema +=   x1 +  2*x2 <=  6
problema +=   x2 -    x1 <=  1
problema +=           x2 <=  2

# Resolvemos y mostramos la cantidad de soluciones
sol = problema.solve()
sol

print('z_max = {0:.4f}, x1 = {1:.4f}, x2 = {2:.4f}'.

Los resultados satisfacen las restricciones del problema. La diferencia x2 - x1 = -1.5 muestra que x2 ser siempre como mximo 1 tonelada mayor que x1.
